●インストール方法

このフォントはWindows用TrueTypeフォントです。以下の方法でインストールして下さい。


++ Windows 95, 98

  1. 「スタート」→「設定」→「コントロールパネル」を開いてください。
  2. 「コントロールパネル」内の「フォント」フォルダを開いてください。
  3. 解凍したフォントファイルを「フォント」フォルダ内にドラッグ＆ドロップしてください。
  4. フォントを使用するアプリケーションを再起動してください。

※フォントを認識してない場合はWindowsを再起動してください。
※Windows 2000, meでの検証はできておりませんが、おそらく使用できます。
